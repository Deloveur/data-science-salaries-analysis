# Data Science Salaries Analysis

## Project Overview

This project presents an Exploratory Data Analysis (EDA) of the **Data Science Salaries** dataset.
The goal of the analysis is to explore salary trends in the data science field and understand how factors such as **experience level, company size, country, and remote work** influence compensation.

The analysis was performed using Python and common data analysis libraries.

---

## Dataset Source

The dataset used in this project was obtained from Kaggle:

Data Science Job Salaries Dataset
https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries

## Analysis Steps

The project includes the following steps:

1. Data loading and inspection
2. Data cleaning and preprocessing
3. Converting country codes to full country names
4. Exploratory Data Analysis (EDA)
5. Data visualization
6. Interpretation of results

---

## Key Questions Explored

The analysis focuses on answering several important questions:

* How does salary change with experience level?
* Which countries offer the highest salaries for data science roles?
* Do larger companies tend to pay higher salaries?
* How common is remote work in data science jobs?
* Are there correlations between numeric variables in the dataset?

---

## Key Insights

The analysis revealed several important insights:

* Salaries increase significantly with experience level.
* Executive-level positions have the highest salaries.
* Some countries offer significantly higher salaries for data science roles.
* Medium and large companies tend to offer higher salaries.
* Remote work is common in data science roles but does not strongly affect salary levels.
