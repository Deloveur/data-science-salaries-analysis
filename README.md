# Data Science Salaries Analysis

This project explores the Data Science Salaries dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Includes
- Salary by experience level
- Salary by country
- Salary by company size
- Remote work distribution
- Correlation heatmap

## Dataset
Data Science Salaries dataset from Kaggle.

## Key Insights
- Salaries increase significantly with experience level.
- Executive-level positions earn the highest salaries.
- Some countries offer higher salaries for data science roles.
- Medium and large companies tend to pay higher salaries.
- Remote work does not strongly affect salary levels in the dataset.
